## 一、选择题（10 题 × 5 分 = 50 分）

**1. （基础）以下哪个是 AI Agent 最核心的特征？**
- A. 能够联网搜索信息
- B. **能够自主感知环境、推理决策并执行行动** ✅
- C. 能够生成文字和图片
- D. 能够记住历史对话

**2. （基础）Agent 与普通 LLM 对话最主要区别是什么？**
- A. Agent 回复速度更快
- B. **Agent 可以调用工具和采取行动** ✅
- C. Agent 不需要输入提示词
- D. Agent 支持更多语言

**3. （基础）以下哪项不属于 Agent 的四大核心组件？**
- A. LLM（大语言模型）
- B. Tools（工具）
- C. Memory（记忆）
- D. **GPU（图形处理器）** ✅

**4. （基础）Agent 中的"工具（Tool）"的作用是什么？**
- A. 美化 Agent 的界面
- B. **让 Agent 能够调用外部功能（如搜索、计算、API）** ✅
- C. 加快 Agent 的响应速度
- D. 给 Agent 提供数据存储

**5. （基础）LLM 在 Agent 中扮演什么角色？**
- A. 输入设备
- B. 存储设备
- C. **"大脑"——负责推理和决策** ✅
- D. 显示设备

**6. （基础）以下哪个描述最符合 Agent 的基本工作流程？**
- A. 输入 → 输出 → 结束
- B. **感知 → 思考 → 行动 → 观察 → 循环** ✅
- C. 提问 → 回答
- D. 训练 → 部署 → 使用

**7. （基础）以下哪个场景最适合使用 Agent？**
- A. 简单的问候回复
- B. **需要多步推理并调用外部工具完成的任务** ✅
- C. 计算 1+1 等于几
- D. 生成一段固定格式的文字

**8. （基础）Agent 中 System Prompt 的主要作用是？**
- A. 让 Agent 自动联网
- B. **设定 Agent 的角色、行为和约束** ✅
- C. 加快 Agent 运行速度
- D. 告诉 Agent 当前时间

**9. （进阶）Function Calling 在 Agent 中的作用是？**
- A. 让 Agent 能听懂用户的语音
- B. **让 Agent 能自动选择并调用合适的工具函数** ✅
- C. 让 Agent 能自我编程
- D. 让 Agent 能保存对话记录

**10. （进阶）以下哪种记忆类型属于 Agent 的短期记忆？**
- A. 向量数据库中存储的知识
- B. **当前对话的上下文** ✅
- C. 训练数据中的知识
- D. 用户长期偏好的存储

## 二、判断题（5 题 × 6 分 = 30 分）

**11. （基础）AI Agent 就是普通的 LLM 聊天对话加上自动回复功能。**
- ❌ **错误** — Agent 的核心特征是自主感知、推理和行动，不是简单的自动回复。

**12. （基础）使用 Agent 必须要有网络连接。**
- ❌ **错误** — Agent 的 LLM 推理可以在本地运行，工具的调用也可能离线完成。

**13. （基础）Agent 在调用工具失败后，可以自动重试或选择其他方案。**
- ✅ **正确** — 这是 Agent 的容错机制之一。

**14. （进阶）Multi-Agent（多智能体）系统中，每个 Agent 都必须使用不同的大语言模型。**
- ❌ **错误** — 多个 Agent 可以共享同一个 LLM，区分在于各自的角色、工具和 Prompt 不同。

**15. （基础）Agent 的 System Prompt 只需要在第一次交互时设定，后续不会影响 Agent 的行为。**
- ❌ **错误** — System Prompt 对 Agent 的每一次交互都有约束作用，持续影响行为。

## 三、填空题（4 题，共 20 分）

**16. （基础，5 分）Agent 的工作循环流程是：感知 → ______ → 行动 → 观察 → 循环**
- ✅ **答案：思考（或 推理 / Thinking）**

**17. （基础，5 分）______ 是 Agent 用来调用外部功能的接口，例如搜索、计算、API 调用等。**
- ✅ **答案：工具（或 Tool / 工具函数）**

**18. （基础，5 分）在 Agent 中，______ 负责将历史对话信息传递给后续轮次，以保证上下文连贯。**
- ✅ **答案：记忆（或 Memory / 记忆模块）**

**19. （进阶，5 分）Agent 使用 ______ 机制来自动判断用户意图并选择合适的工具函数进行调用。**
- ✅ **答案：Function Calling（或 函数调用 / 工具调用）**
